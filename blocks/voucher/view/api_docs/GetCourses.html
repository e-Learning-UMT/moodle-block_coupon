VoucherAPI - GetCourses

Get all non-sidewide and visible courses.

Method signature

GetCourses()

Return

typedescription
arrayList of courses.

Return array can or will contain the following parameter keys:

nametypedescription
$idintCourse ID
$fullnamestringCourse fullname

Example code / usage

 $url = 'wwwroot?>/blocks/voucher/view/api.php?method=GetCourses&resultType=xml';
 
 $params = array(
 'username' => '{API username}',
 'password' => '{API password}'
 );
 
 $ch = curl_init($url);
 curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
 curl_setopt($ch, CURLOPT_USERPWD, $params['username'].':'.$params['password']);
 $result = curl_exec($ch);
 
 echo htmlspecialchars($result);
